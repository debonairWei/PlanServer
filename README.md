水边开发的计划任务管理服务
==================

1、类似于Windows的计划任务，可以添加管理定时运行的任务，或不允许退出的任务；
2、按可执行文件路径作为唯一key，避免多次启动
3、支持服务 或 可执行文件方式
4、支持web管理
